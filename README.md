# Tests Platform
## Description 
On the website users can create and pass quizzes.  

## Functionality 
- create quiz
- pass quiz
  
## Technology stack 
Client: React js, Typescript, ReduxToolkit, RTKQuery

Server: Node js, Express js, Typescript, mongoose

Database: Mongo DB


## New knowledges 
- RTK + typescript
- RTKQuery
